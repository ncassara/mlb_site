# mlb_site

A simple PHP/MySQL web application that manages MLB data with role-based access.

**Features**
- Login system with three roles: fan, coach, admin

- Dashboards for each user role

- View teams and players

- Coaches can add players

**Getting Started**
1. Install MAMP and start Apache & MySQL.

2. Place the project folder in MAMP/htdocs.

3. Import the SQL files (create.sql, load.sql, login.sql) into a new database using phpMyAdmin.

4. Update config.php with your database credentials.

5. Open in browser: http://localhost:8888/mlb_site
